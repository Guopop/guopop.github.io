# 一些 Apple 链接

[Apple Developer](https://developer.apple.com) 苹果官方开发平台，开发，设计，所有文档。

[Swift](https://www.swift.org) Swift官网。

[Create With Swift](https://www.createwithswift.com) 关于swiftui开发的网站，有很多设计方面的demo，让SwiftUI开发更有艺术性。

[WWDC Notes](https://wwdcnotes.com) 汇总历届WWDC内容。

[肘子的 Swift 记事本](https://fatbobman.com/zh/) 深度讲解SwiftUI、SwiftData内容的博客。

[iOS Resolution](https://www.ios-resolution.com) 关于iPhone、iPad、Apple Watch设备规格的网站。

[SwiftUI Field Guide](https://www.swiftuifieldguide.com) 生动讲解 SwiftUI 特性的网站，有代码，可实时调整效果。

[Apple iOS 26 UI Kit](https://www.sketch.com/s/f63aa308-1f82-498c-8019-530f3b846db9/symbols) Apple iOS 26 sketch 设计资源，颜色，字体...

[LYon's Blog](https://yindongliang.com/docs/Apple/) 一个简略汇总了 Apple 技术的博客。

[Captain SwiftUI](https://captainswiftui.substack.com) 深度讲解 SwiftUI 的网站。

[Apple Slides](https://appleslides.com) 苹果发布会海报。

[Design Code](https://designcode.io/) 一个 SwiftUI 教程网站，还有Figma、UI等教程，设计向。

[SwiftUIX](https://swiftuix.com) 一个 SwiftUI、UIKit、Swift 教程网站，中文，详细。

[AppCoda](https://www.appcoda.com) 一个 SwiftUI 教程网站。

[StewartLynch YouTube](https://stewartlynch.github.io/StewartLynchChannel/) 一个深入讲解 SwiftUI 的 Youtuber 的视频目录。

[Explore SwiftUI](https://exploreswiftui.com) 探索 SwiftUI Api 使用的一个网站，有代码演示。
